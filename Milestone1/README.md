# Milestone 1
# Supply Chain Visibility & Optimization - Milestone 1

## Objective
Build a star schema data model in Power BI for supply chain analysis.

## Dataset Source
https://www.kaggle.com/datasets/shashwatwork/dataco-smart-supply-chain-for-big-data-analysis

## Data Cleaning and Transformation
- Removed unnecessary columns
- Checked and handled missing values
- Removed duplicates from dimension tables
- Corrected data types
- Created Customer, Product, Category, Shipping, Department, Location and Date dimensions
- Built a star schema

## Data Model Overview
- Fact Table
- Customer Dimension
- Product Dimension
- Category Dimension
- Department Dimension
- Shipping Dimension
- Location Dimension
- Date Dimension

Relationships are One-to-Many with the fact table.

## Tools Used
- Power BI Desktop
- Power Query Editor
- DAX
